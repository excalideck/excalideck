**Excalideck** is an app for authoring slide decks that look hand-drawn.
**Excalideck** is based on the awesome
[**Excalidraw**](https://excalidraw.com/).

## Try it

Go to [excalideck.com](https://excalideck.com) to start drawing a deck.
